ArcFlow — Smart Value Router

ArcFlow is a simple MVP (Minimum Viable Product) that demonstrates how value can be routed intelligently across chains using basic decision logic.

🚀 What This App Does

- Connect wallet
- Enter amount and recipient
- Automatically decides the route:
  - Small amount → Direct send
  - Large amount → Bridge route (CCTP concept)

This is a frontend-only demo built for learning and experimentation.

---

🧠 Concept

Instead of users deciding:

- which chain to use
- how to bridge
- how to send

ArcFlow introduces a simple idea:

👉 Let the app decide the best route.

This is the first step toward automated crosschain value routing.
